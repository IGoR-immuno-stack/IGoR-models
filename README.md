# igor-models

Model files for the IGoR (Inference and Generation of Repertoires) immunogenomic analysis tool.

## Contents

- `human/` - Human immune receptor models
- `mouse/` - Mouse immune receptor models  
- `heavy_pen_substitution_matrix.csv` - Substitution matrix for heavy chain analysis

## Usage

These models are typically used as a git submodule in the main IGoR repository:

```bash
git submodule add https://gitlab.inria.fr/ant-men/igor-stack/igor-models.git models
```
